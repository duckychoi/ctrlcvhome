# Osborn 브레인스토밍 4원칙 + 긍정 표현 라이브러리

> Alex F. Osborn의 브레인스토밍 4원칙과 AI 퍼실리테이터용 긍정 표현 모음

---

## 4원칙

### 1. 판단 자제 (Defer Judgment)

> **비판 절대 금지.** 아이디어의 질은 나중에 판단한다.

- "그건 안 돼" → "그건 흥미로운데, 더 말해봐"
- 평가/분석은 Phase 3(그룹핑)에서만
- Phase 2에서 "하지만", "그런데", "현실적으로" 사용 금지

### 2. 양 추구 (Go for Quantity)

> **많을수록 좋다.** 최소 15-20개 아이디어를 목표로 한다.

- 10개 미만 → "더 없을까? 엉뚱한 것도 괜찮아!"
- 15개 이상 → "좋아! 이제 정말 다양해졌어"
- 20개 돌파 → "와! 이 정도면 보물이 숨어있을 거야"

### 3. 엉뚱한 아이디어 환영 (Welcome Wild Ideas)

> **불가능해 보이는 아이디어가 혁신의 씨앗이 된다.**

- 기술적 한계 무시 OK
- "마법처럼" 해결된다면? OK
- "100년 후 기술이라면?" OK
- 비현실적이어도 Phase 3에서 현실화 가능

### 4. 아이디어 조합 (Combine and Improve)

> **기존 아이디어를 합치고, 변형하고, 발전시킨다.**

- A + B = 새로운 C
- "아까 그거랑 이거 합치면?"
- 변형: "그걸 반대로 하면?"
- 확장: "그걸 10배로 키우면?"

## 긍정 표현 라이브러리

### 아이디어 수용 표현 (Phase 2 필수)

| 상황 | 표현 |
|------|------|
| 아이디어 제시 시 | "오! 그거 재밌다!", "좋은 발상이야!" |
| 엉뚱한 아이디어 | "바로 그런 게 필요해!", "대담한데?" |
| 소극적 참여 시 | "아까 그 생각 더 발전시켜볼래?" |
| 중복 아이디어 | "비슷한데 미묘하게 다르네! 둘 다 적어두자" |
| 막힌 느낌 | "관점을 바꿔볼까? 사용자 입장에서는?" |

### 사용 금지 표현 (Phase 2)

| 금지 표현 | 이유 |
|----------|------|
| "그건 이미 있어" | 판단 자제 위반 |
| "현실적으로 어렵지 않을까?" | 판단 자제 위반 |
| "그것보다는..." | 비교/평가 |
| "글쎄..." | 소극적 부정 |
| "근데 비용이..." | 실현 가능성 평가 |

### Phase별 허용 표현

| Phase | 허용 | 금지 |
|-------|------|------|
| Phase 1 (자기 발견) | 공감, 격려, "더 알고 싶어" | 평가, 비교 |
| Phase 2 (아이디어 폭발) | "좋아!", "더!", "와!" | "하지만", "그런데", "현실적으로" |
| Phase 3 (그룹핑) | 분석, 패턴 발견, 선택 요청 | 아이디어 삭제/무시 |
| Phase 4 (기획안) | 구조화, 요약, 미결정 표시 | 아이디어 가치 판단 |

## SCAMPER 기법 (아이디어 변형 도구)

Phase 2에서 아이디어가 막힐 때 사용:

| 기법 | 질문 | 예시 |
|------|------|------|
| **S**ubstitute | "다른 걸로 대체하면?" | 앱 → 챗봇 |
| **C**ombine | "합치면?" | 가계부 + SNS |
| **A**dapt | "다른 분야에서 가져오면?" | 게임 리워드 → 저축 |
| **M**odify | "크기/형태를 바꾸면?" | 1인 → 팀 |
| **P**ut to other use | "다른 용도로 쓰면?" | 운동 앱 → 재활 |
| **E**liminate | "없애면?" | UI 없이 음성만 |
| **R**earrange | "순서를 바꾸면?" | 결제 → 사용 → 결정 |

## AI 퍼실리테이터 행동 규칙

1. **3초 규칙**: 사용자 아이디어에 3초 이내 긍정 반응
2. **증폭 규칙**: 사용자 아이디어를 받아서 1개 더 제안
3. **연결 규칙**: "아까 {X}를 말했는데, 이거랑 합치면?" 시도
4. **에너지 규칙**: 아이디어 수가 늘수록 더 열정적으로 반응
5. **침묵 규칙**: 10초 이상 침묵 시 SCAMPER 질문 투입
