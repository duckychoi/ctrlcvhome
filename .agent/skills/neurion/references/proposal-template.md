## 생성 규칙

### 필수 포함 항목

| 항목 | 필수 여부 | 설명 |
|------|----------|------|
| 핵심 아이디어 한 줄 | 필수 | 방향성 요약 |
| 전체 아이디어 목록 | 필수 | 모든 아이디어 (삭제 없이) |
| 그룹 분류 | 필수 | 최소 2개 그룹 |
| 선택된 방향 | 필수 | 사용자 선택 결과 |
| 핵심 기능 3개 | 필수 | 초안 수준 (변경 가능) |
| 미결정 사항 | 필수 | /socrates에서 결정할 것 |

### 작성 원칙

1. **아이디어 삭제 금지**: 모든 아이디어를 기록 (나중에 활용 가능)
2. **출처 표시**: 사용자/AI 구분 (누가 제안했는지)
3. **판단 배제**: "좋은/나쁜" 평가 없이 사실만 기록
4. **미결정 명시**: 아직 결정되지 않은 것을 명확히 표시
5. **다음 단계 안내**: /socrates 연결 명시

---

## /socrates 연동

`neurion-proposal.md`가 존재할 때 /socrates의 활용 방식:

```
1. 핵심 기능 3개 → Phase 1의 초기 기능 시드로 활용
2. 타겟 사용자 → 질문 맥락에 반영
3. 미결정 사항 → 해당 Phase에서 우선 질문
4. 전체 아이디어 목록 → 추가 기능 탐색 시 참조
```

**중요**: Socratic 검증은 그대로 수행. 기획안의 내용을 무조건 수용하지 않고, 질문을 통해 검증/발전시킴.
