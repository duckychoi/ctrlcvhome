# 3단계 자기 발견 프레임워크

> Phase 1 (선택): 사용자가 아이디어를 정하지 못했을 때 사용하는 자기 발견 프로세스

---

## 개요

```
단계 1: 성취 발견  →  "뭘 잘하고, 뭘 좋아하세요?"
단계 2: 역량 매핑  →  "그 경험에서 어떤 능력이 드러나요?"
단계 3: 적용 탐색  →  "그 능력으로 뭘 만들 수 있을까?"
```

**사용 조건**: Phase 0에서 사용자가 "아이디어 없음"을 선택한 경우에만 실행

---

## 단계 1: 성취 발견

### 질문 세트

```json
{
  "questions": [{
    "question": "🎯 아이디어를 찾기 전에, 당신에 대해 조금 알고 싶어요.\n\n최근 1년간 가장 뿌듯했던 경험이 뭔가요?\n\n(일, 취미, 학습, 관계 뭐든 OK!)",
    "header": "성취 발견",
    "options": [
      {"label": "일/직장에서", "description": "프로젝트, 성과, 문제 해결 등"},
      {"label": "취미/관심사에서", "description": "운동, 창작, 게임, 요리 등"},
      {"label": "학습/성장에서", "description": "새로운 기술, 자격증, 언어 등"},
      {"label": "관계/커뮤니티에서", "description": "모임, 봉사, 네트워킹 등"}
    ],
    "multiSelect": false
  }]
}
```

### 후속 질문 (선택에 따라)

```
사용자: "취미에서요. 요리를 시작했는데 레시피를 기록하고 있어요"

🎯 "오! 요리와 기록이라... 그 과정에서 가장 재미있는 순간은 언제예요?"

→ 핵심 동기 추출: "기록", "공유", "실험", "완성" 중 하나
```

---

## 단계 2: 역량 매핑

### 질문 세트

```json
{
  "questions": [{
    "question": "🎯 {성취 경험}에서 이런 능력을 발휘한 것 같아요.\n\n💡 제가 보기에:\n• {역량1}: {근거}\n• {역량2}: {근거}\n• {역량3}: {근거}\n\n이 중에서 가장 자신 있는 게 뭔가요?",
    "header": "역량 매핑",
    "options": [
      {"label": "{역량1}", "description": "{설명}"},
      {"label": "{역량2}", "description": "{설명}"},
      {"label": "{역량3}", "description": "{설명}"}
    ],
    "multiSelect": false
  }]
}
```

### 역량 추출 예시

| 성취 | 추출 역량 |
|------|----------|
| 요리 레시피 기록 | 체계적 정리, 실험 정신, 시각적 표현 |
| 운동 루틴 관리 | 습관 형성, 데이터 추적, 자기 관리 |
| 모임 운영 | 커뮤니티 관리, 일정 조율, 소통 |
| 사이드 프로젝트 | 문제 발견, 기술 학습, 완성 능력 |

---

## 단계 3: 적용 탐색

### 자동 아이디어 시드 생성

역량 + 관심사를 조합하여 브레인스토밍 시드를 자동 생성:

```
역량: 체계적 정리 + 관심사: 요리

→ 자동 시드:
  1. "요리 레시피 정리 앱"
  2. "재료 기반 추천 서비스"
  3. "요리 실험 기록 플랫폼"
  4. "초보 요리사 가이드"
  5. "식재료 관리 도구"
```

### 사용자에게 제시

```json
{
  "questions": [{
    "question": "🎯 당신의 강점({역량})과 관심사({관심사})를 조합해봤어요!\n\n💡 이런 방향들이 떠올라요:\n1. {시드1}\n2. {시드2}\n3. {시드3}\n4. {시드4}\n5. {시드5}\n\n이 중에 끌리는 게 있나요? 아니면 완전 다른 것도 OK!",
    "header": "방향 선택",
    "options": [
      {"label": "{시드1}", "description": "{간단 설명}"},
      {"label": "{시드2}", "description": "{간단 설명}"},
      {"label": "다른 방향", "description": "위에 없는 새로운 아이디어"}
    ],
    "multiSelect": false
  }]
}
```

---

## Phase 2로 전환

자기 발견 완료 후:

```
🎯 "좋아요! '{선택된 시드}'를 기반으로 브레인스토밍을 시작할게요!
    이제부터는 판단 없이, 양으로 승부! 어떤 아이디어든 환영이에요!"

→ Phase 2 (아이디어 폭발) 시작
→ 선택된 시드를 초기 주제로 사용
```

---

## 전체 시간 가이드

| 단계 | 예상 시간 | AskUserQuestion 횟수 |
|------|----------|---------------------|
| 성취 발견 | 1-2분 | 1-2회 |
| 역량 매핑 | 1분 | 1회 |
| 적용 탐색 | 1-2분 | 1회 |
| **합계** | **3-5분** | **3-4회** |
