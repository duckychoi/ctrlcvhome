# Serendipity: 창의적 연결 도메인

## 개요

사용자의 아이디어를 예상치 못한 분야와 연결하여 창의적 확장을 유도하는 기능입니다.
Q7(핵심 기능) 완료 후 **랜덤하게 1개 분야**를 선택하여 제안합니다.

---

## 8가지 연결 도메인

| # | 도메인 | 영문 | 핵심 관점 |
|---|--------|------|----------|
| 1 | 역사 | History | 과거의 유사 사례, 역사적 교훈, 시대별 변화 |
| 2 | 산업 | Industry | 다른 산업의 성공 모델, 업종 간 융합 |
| 3 | 인문학 | Humanities | 인간 본성, 철학적 관점, 윤리적 고려 |
| 4 | 트렌드 | Trends | 최신 기술/사회 트렌드, Z세대 문화, 미래 예측 |
| 5 | 과학 | Science | 과학적 원리, 자연에서의 영감, 연구 결과 |
| 6 | 예술 | Art | 예술적 표현, 미학, 감각적 경험 |
| 7 | 문학 | Literature | 스토리텔링, 서사 구조, 문학적 은유 |
| 8 | 음악 | Music | 리듬, 조화, 감정적 공명, 음악 산업 |

## 도메인별 연결 질문

### 1. 역사 (History)

**연결 프롬프트:**
```
흥미로운 관점을 제안드려도 될까요?

당신의 아이디어를 '역사'의 관점에서 생각해봤어요.

예를 들어:
- 과거에 비슷한 문제를 해결한 사례가 있었을까요?
- 역사 속 어떤 도구/시스템에서 영감을 받을 수 있을까요?
- 시대가 변하면서 이 문제는 어떻게 진화했나요?

{프로젝트명}을 역사적 관점에서 연결해본다면,
어떤 시대나 사건이 떠오르시나요?
```

**예시 연결:**
- 가계부 앱 → 조선시대 가계부(장책), 르네상스 시대 상인들의 복식부기
- 헬스케어 앱 → 고대 그리스 히포크라테스의 예방의학
- 협업 도구 → 중세 길드의 협업 시스템

예를 들어:

- 게임 산업의 보상 시스템이 교육에 적용된 것처럼

{프로젝트명}에 적용할 수 있는 다른 산업의 성공 모델이 있을까요?
어떤 업종에서 영감을 받고 싶으신가요?

**예시 연결:**

- 학습 앱 → 게임 산업의 레벨업/업적 시스템
- 쇼핑몰 → 물류 산업의 실시간 추적 시스템

## 실행 규칙

### 발동 시점

**Q7(핵심 기능) 완료 직후** 세렌디피티 기능을 실행합니다.

```
Q7 답변 수집 완료
    ↓
랜덤으로 1개 도메인 선택
    ↓
AskUserQuestion으로 세렌디피티 제안
    ↓
사용자 응답에 따라 처리
    ↓
Q8로 진행
```

### 랜덤 선택 방식

```python
# 의사 코드
import random
domains = ["역사", "산업", "인문학", "트렌드", "과학", "예술", "문학", "음악"]
selected_domain = random.choice(domains)
```

### AskUserQuestion 형식

```json
{
  "questions": [{
    "question": "잠깐, 재미있는 아이디어가 떠올랐어요!\n\n당신의 '{프로젝트명}'을 '{선택된 도메인}'의 관점에서 생각해보면 어떨까요?\n\n{도메인별 연결 프롬프트}\n\n이 관점에서 떠오르는 아이디어가 있으신가요?",
    "header": "창의적 연결",
    "options": [
      {"label": "흥미로워요!", "description": "이 관점에서 아이디어를 더 탐구"},
      {"label": "다른 분야로", "description": "다른 분야의 관점이 궁금해요"},
      {"label": "건너뛰기", "description": "원래 질문으로 돌아가기"}
    ],
    "multiSelect": false
  }]
}
```

### 응답 처리

| 사용자 응답 | 처리 |
|------------|------|
| "흥미로워요!" | 해당 도메인 심화 질문 후 Q8로 |
| "다른 분야로" | 다른 도메인 1개 랜덤 제안 (최대 1회 추가) |
| "건너뛰기" | 바로 Q8로 진행 |
| Other (직접 입력) | 사용자 아이디어 기록 후 Q8로 |

## 세렌디피티 결과 활용

사용자가 세렌디피티 질문에 답변한 경우, 해당 내용을 기획 문서에 반영합니다.

### PRD에 반영

```markdown
## 창의적 영감 (Serendipity Insights)

- **연결 도메인**: {선택된 도메인}
- **사용자 인사이트**: {사용자 답변}
- **적용 가능성**: {AI 분석}
```

### 예시

```markdown
## 창의적 영감 (Serendipity Insights)

- **연결 도메인**: 음악
- **사용자 인사이트**: "식단 기록도 음악처럼 리듬이 있으면 좋겠어요.
  아침-점심-저녁의 규칙적인 비트처럼요."
- **적용 가능성**: 식단 기록 완료 시 경쾌한 사운드 피드백,
  주간 리포트를 음악적 시각화(높은 칼로리 = 높은 음, 낮은 칼로리 = 낮은 음)로 표현
```

## 비활성화 조건

다음 경우 세렌디피티 기능을 건너뜁니다:

1. 사용자가 "빠르게 진행해주세요"라고 요청한 경우
2. 이미 Q1~Q7에서 창의적 연결 아이디어를 많이 언급한 경우
3. 시간 제한이 있는 세션인 경우
